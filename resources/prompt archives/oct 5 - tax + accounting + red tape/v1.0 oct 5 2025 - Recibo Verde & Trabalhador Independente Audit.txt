### **2. Quarterly and Annual Workflow**

**Every Quarter (Jan – Apr – Jul – Oct):**

1. Login → [Segurança Social Direta → Trabalhador Independente → Declaração Trimestral]
2. Enter income from recibos verdes only (do not include Xolo/Stripe earnings).
3. If zero, enter “0” → system keeps minimum ~€20.
4. Pay within month following the quarter (e.g., April 30 for Q1).
5. Download payment proof.

**Every Year (Apr – Jun):**

1. File IRS (Modelo 3) via Portal das Finanças.
2. Include recibo verde income auto-imported from e-Fatura.
3. If no income, submit €0 declaration to stay active.

### **3. Issuing Recibos Verdes (E-Fatura Portal)**

**Use only for Portugal-based clients.**

Foreign invoices must go through Xolo or Stripe.

**Steps:**

1. Portal das Finanças → *Emitir Recibo Verde Eletrónico*
2. Tipo de Rendimento → “Prestação de Serviços”
3. Descrição → brief service summary (“Consultoria, serviços digitais”)
4. IVA → “Isento Art. 53º CIVA”
5. Retenção na Fonte → choose option:
    
    • 23 % (default 2025)
    
    • 25 % (optional higher)
    
    • None (if previous year ≤ €15 000)
    
6. Save PDF for records.

✅ 3–5 receipts/year from different clients = active low-risk.

❌ >10 receipts to same client = dependency risk.

1. Portal das Finanças → *Declaração de Cessação*.
